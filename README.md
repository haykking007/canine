## About the project
Canine is an easy to use intuitive deployment platform for Kubernetes clusters.

## Requirements

* Docker v24.0.0 or higher
* Docker Compose v2.0.0 or higher

## Installation

```git clone git@github.com:czhu12/canine.git
cd canine
docker compose up
```
